# Muhammad Ali Sheikh - Personal Resume

## Project Overview
A professional personal resume website showcasing academic background, skills, experience, projects, and certifications. Built with pure HTML and inline CSS for a clean, responsive design.

## Features
- Responsive layout optimized for desktop and mobile devices
- Professional design with structured sections
- Contact information and social links
- Academic achievements and GPA display
- Project portfolio highlights
- Certification listings

## Tech Stack
- HTML5
- CSS3 (inline styles)

## Setup & Usage
1. Clone the repository
2. Open `index.html` in a web browser
3. No additional dependencies required

## GitHub Pages Deployment
1. Push code to a GitHub repository
2. Go to repository Settings > Pages
3. Select "Deploy from a branch" and choose main branch
4. Access the live site at `https://[username].github.io/[repository-name]`
