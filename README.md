## Project Overview

A professional personal resume website built with pure HTML and inline CSS. This responsive portfolio demonstrates academic achievements, technical skills, work experience, projects, and certifications in a clean, mobile-optimized design.

### Key Highlights
- Clean, modern UI with professional typography
- Fully responsive across all devices
- Fast loading with no external dependencies
- Secure and lightweight implementation

## Setup & Usage

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software required

### Local Development
1. **Clone the repository**
   ```bash
   git clone https://github.com/alisheikh55209-png/HTML-CSS-Resume.git
   cd HTML-CSS-Resume
   ```

2. **Open in browser**
   - Double-click `index.html` or
   - Right-click `index.html` ‚Üí Open with ‚Üí Your browser

3. **No dependencies needed** - Works immediately!

## üöÄ GitHub Pages Deployment

### Automatic Deployment
1. Fork this repository
2. Go to **Settings** ‚Üí **Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose **main** branch and **/(root)** folder
5. Click **Save** - Your site will be live at `https://[username].github.io/HTML-CSS-Resume/`

### Manual Deployment
1. Push your changes to GitHub
2. Enable GitHub Pages as described above
3. Wait 2-3 minutes for deployment
